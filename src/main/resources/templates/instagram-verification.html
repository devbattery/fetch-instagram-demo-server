Instagram Verification Demo

인스타 인플루언서 인증을 시뮬레이션합니다.

실제 서비스에서는 이 단계에서 인스타그램 로그인 또는 계정 연결이 끝난 뒤 딥링크나 유니버설 링크로 앱으로 복귀하게 됩니다. 이 데모는 그 마지막 복귀 단계를 빠르게 검증하기 위한 화면입니다.

핸들
@demo_handle
프로필 URL
https://www.instagram.com/demo_handle/
복귀 URL
demowebview://verification/callback
공식 API 기준으로는 "파란 체크 뱃지 여부"를 단순 외부 프로필 URL만으로 바로 판정하는 안정적인 흐름을 기대하기 어렵기 때문에, 이 데모는 인증 완료 후 앱으로 success code를 돌려보내는 구조만 검증합니다.

또한 실제 인스타 인플루언서 계정이 없어도 테스트할 수 있습니다. 지금 화면은 실제 계정 검증이 아니라, 인증 성공/실패 후 앱 복귀 처리가 제대로 되는지만 보는 QA 페이지입니다.

네이티브 앱에서는 위 복귀 URL을 custom scheme 또는 universal link로 바꾸면 됩니다.
